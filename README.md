# Text to PNG Converter Setup

## 1. Font Installation
This application requires the **Samsung Sharp Sans** font to look its best.
Since this font is proprietary, we cannot include it directly.

**Please follow these steps:**
1.  Search for "Samsung Sharp Sans font download" or locate your font file.
2.  Rename the font file to `SamsungSharpSans.ttf` (or `SamsungSharpSans-Bold.ttf`).
3.  Place the font file(s) inside the `fonts` folder in this directory.

Supported filenames in transparency:
- `fonts/SamsungSharpSans.ttf`
- `fonts/SamsungSharpSans-Bold.ttf`

## 2. Usage
- Open `index.html` in any modern web browser.
- Type your text (supports English, Hindi, Korean, etc.).
- Convert to PNG and download!
